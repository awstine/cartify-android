# Cartify

This repository contains multiple Cartify modules, including:

- `backend/` for the backend API
- `Admin/` for the web admin and storefront frontend
- `app/` for the Android application

The full web-platform documentation is in [DOCUMENTATION.md](DOCUMENTATION.md).

## Documentation Scope

`DOCUMENTATION.md` is the primary technical document for:

- the `backend/` service
- the `Admin/` web application
- architecture, APIs, data model, setup, deployment, operations, and security

The Android application is intentionally excluded from that document unless noted briefly for repository context.

## Quick Start

Backend:

```bash
cd backend
npm install
npm run dev
```

Web frontend:

```bash
cd Admin
npm install
npm run dev
```
